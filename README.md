
# Pentominoes

## Purpose of Project:

The Pentominoes project is a recreational and educational software application that explores the world of pentomino puzzles. Pentominoes are a type of polyomino puzzle where you aim to fit a set of 12 unique shapes made up of five squares each into a larger grid. The project provides an interactive platform for solving these puzzles, enhancing spatial reasoning, and having fun.
